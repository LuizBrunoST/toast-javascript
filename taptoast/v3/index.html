TapToast v3

Documentação completa com demonstrações

Introdução

TapToast v3 é uma biblioteca de notificações Toast em Vanilla JavaScript. Leve, rápida e sem dependências externas.

Recursos principais:

  • 100% Vanilla JS
  • CSS próprio incluso
  • Configuração global (posição e duração)
  • Empilhamento automático
  • Fácil de integrar em qualquer projeto

Instalação

Inclua os arquivos no seu projeto:

<link rel="stylesheet" href="taptoast.css">
<script src="taptoast.js"></script>

Configuração

Defina a posição e duração padrão:

TapToast.config({
    position: "top-right", // top-right | top-left | bottom-right | bottom-left
    duration: 3000 // tempo em ms
});

Uso Básico

TapToast.success("Sucesso!");
TapToast.error("Erro!");
TapToast.warning("Aviso!");
TapToast.info("Informação!");
