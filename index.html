TapToast Documentation

Notificações Toast em JavaScript — Todas as versões

Versão Atual: v3 latest

Novidades e explicação completa:

  • 100% Vanilla JS — sem dependências
  • CSS próprio incluso
  • Configuração global (posição e duração)
  • Empilhamento automático de notificações
  • Leve e fácil de integrar em qualquer projeto

Instalação

<link rel="stylesheet" href="taptoast/v3/taptoast.css">
<script src="taptoast/v3/taptoast.js"></script>

Configuração

TapToast.config({
    position: "top-right", // top-right | top-left | bottom-right | bottom-left
    duration: 3000
});

Uso

TapToast.success("Sucesso!");
TapToast.error("Erro!");
TapToast.warning("Aviso!");
TapToast.info("Informação!");
